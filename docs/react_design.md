Agent State
What information does the agent maintain throughout execution?
The ReAct agent maintains a comprehensive record of the session, often referred to in the sources as the context or trajectory
. Based on the sources, the information maintained includes:
User Query: The initial task, question, or claim provided at the start of the session
.
Current Prompt: The foundational instructions, which include human-written "exemplars" (few-shot in-context examples) that teach the model the interleaved format of thinking and acting
.
Previous Thoughts: Verbal reasoning traces generated by the model to decompose goals, track progress, or incorporate commonsense
.
Previous Actions: Task-specific commands issued to the environment, such as Wikipedia searches or household navigation commands
.
Previous Observations: External feedback received from the environment or tools (like Wikipedia text or room state updates) after an action is performed
.
Current Step Number: While implicit in the trajectory, the step count is tracked and used to trigger termination heuristics if the task takes too long (e.g., stopping after 7 steps for certain tasks)
.
Final Answer: The definitive result produced when the model decides to trigger the finish[answer] action
